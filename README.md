# Rabobank-labtest

## What is the use of this Repo

This Project is a Simple Rabobank Lab Test
## Prerequisites

### Install Node JS
Refer to https://nodejs.org/en/ to install nodejs

Clone the project into local

Go to frontend dir

Install all the npm packages.

```bash
npm install
```

```bash
npm run start
```

Go to backend dir

Install all the npm packages.

```bash
npm install
```

```bash
node server.js
```

#### Front-end URL

http://localhost:3000/
